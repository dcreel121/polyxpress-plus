PolyXpress has moved!

To allow for improved application scalability, the static web pages associated with PolyXpress have moved to mhaungs.github.com/PolyXpress. The move also provides a central point-of-access for PolyXpress and improves the development workflow.

Previously, PolyXpress could be accessed at either px.cfapps.io (production version) and pxdev.cfapps.io (development version). The two sites were completely separate and did not share any data. Now, the site mhaungs.github.com/PolyXpress only provides links to the production version. This version is more stable and will not change frequently.

If you need to access your work that you saved on pxdev.cfapps.io (the development version), you can still access them at mhaungs.github.com/PolyXpress.

Ways to Contribute!!

  • PolyXpress now has a public Wiki. Please add to our wiki to add tutorial information, provide best practices, or better explain PolyXpress concepts and workflow.
  • PolyXpress now has an issue tracker. If you notice an error in PolyXpress or would like to request a feature, you can go to the PolyXpress Issue Tracker and submit your issue.
